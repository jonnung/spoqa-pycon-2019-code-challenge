# ASCII-SCROLL: CONNECT THE PYTHONISTAS
## 설치
### 파이썬 가상환경 설정

> 요구 사항: Python3

```
$ python3 -m venv venv
$ source venv/bin/activate
```

### 의존성 패키지 설치
```
$ pip install -r 
```

## 실행
```
$ python src/run.py
```
